Display your chat widget from <a href="http://meebome.com">meebo me</a>. The code is perfectly XHTML valid and won't break the validation of your site.


== Installation ==

1. Download MeeboMe Plugin. 
2. Extract the contents of meebo-me.zip to wp-content/plugins/ folder. You should get a folder called meebo-me.
3. Activate the Plugin in WP-Admin. 
4. Goto Options > meebo me. If you are upgrading from an earlier version, it is recommended that you click Default Options so as to clean up any old options. 
5. Add <?php if (function_exists('echo_meebome')) echo_meebome(); ?> where you want to display the meebo me widget.

== Frequently Asked Questions ==

= What are the requirements for this plugin? =

WordPress 1.5 or above


= Can I customize what is displayed? =

Yes you can. Visit Options > meebo me
1. Widget ID - This is the 10 character ID in src attribute of the embed tag. e.g. In <embed src="http://widget.meebo.com/mm.swf?vvjWAFUxUI" the ID is <strong>vvjWAFUxUI</strong>.
2. Size - Select the size of the Meebo Widget. 


For more information, please visit http://ajaydsouza.com/wordpress/plugins/meebo-me-plugin/#customizing

= Do I really need this plugin? =
The code generated by meebo me is not valid XHTML. If you want to display the meebo widget using perfectly valid XHTML code, you can use this plugin.
